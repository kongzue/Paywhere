# Paywhere
花哪儿记账相关资料

## 简介
花哪儿是一款专注记账的工具
无广告、无后台、无闪屏、无隐私权限
已率先通过「Android 绿色应用公约」认证

【个人账单】
一拉一点，3.5 秒轻松记账
以极度克制的功能，并在界面和交互方式上化繁为简，让每一次使用都不费吹灰之力

【多人账单】
大家一起用，记录在同一个账单内
实时统计消费、自动 AA 计算
用于情侣同居、室友合租、同事轮流点餐、驴友出行游玩等

毫不克制的
- 支持账单云同步
- 支持 2K 分辨率设备
- 支持 Ticwatch 手表记账
- 支持 One Step 一步记账
- 支持 Android7.0 Shortcuts
